# Spring Boot Asynchronous Tasks Example

Цей проект демонструє використання асинхронних задач у Spring Boot за допомогою анотацій `@Scheduled` та `ScheduledExecutorService`. У програмі реалізовані два завдання:

1. Кожні 10 секунд оновлюється файл з лічильником, який збільшується на 5.
2. Через 5 секунд після запуску програми виводиться повідомлення в консоль.

## Вимоги

Для запуску проекту вам потрібно:

- Java 17 або новіша
- Maven для збірки та управління залежностями
- Spring Boot

## Залежності

У проекті використовуються наступні залежності:

- `spring-boot-starter` — для основної роботи з Spring Boot.
- `spring-boot-starter-scheduling` — для підтримки планування задач.

## Як запустити

### 1. Клонуйте репозиторій

### 2. Зберіть проект

```bash
mvn clean install
```

### 3. Запустіть програму

```bash
mvn spring-boot:run
```

### 4. Перевірте результат

1. Кожні 10 секунд буде оновлюватися файл `counter.txt` з новим значенням лічильника, який збільшується на 5.
2. Через 5 секунд після запуску програми виводиться повідомлення: `5 секунд від запуску програми`.

## Структура проекту

### Основні класи:

- **FileUpdater**: Клас, який оновлює значення лічильника в файлі кожні 10 секунд.
- **MessagePrinter**: Клас, який виводить повідомлення в консоль через 5 секунд після запуску програми.
- **AsyncApplication**: Головний клас, який запускає Spring Boot додаток.

### Принцип роботи:

- Використовується анотація `@Scheduled` для планування задач з інтервалами:
    - `fixedRate = 10000` — виконується кожні 10 секунд.
    - `initialDelay = 5000` — затримка перед виконанням на 5 секунд.
- Планувальник запускається через `@EnableScheduling` у головному класі.
